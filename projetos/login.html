Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprimorar habilidades em Front-End, explorando responsividade, validação de formulários e boas práticas de desenvolvimento web. A proposta é entregar uma interface limpa, funcional e intuitiva em diferentes dispositivos.

  • CSS3
  • JavaScript
